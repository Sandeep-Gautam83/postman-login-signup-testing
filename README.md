# postman-login-signup-testing
This project contains API testing collections for Login and Signup authentication modules using Postman. It is designed to test user authentication endpoints such as user registration, login, token validation, and error handling.

The collection helps developers and testers easily verify API responses, request validation, authentication flow, and backend integration.

Features
User Signup API Testing
User Login API Testing
Request & Response Validation
JWT Token Testing
Error Handling Test Cases
Environment Variables Support
Easy API Import & Execution

Technologies Used
Postman
Java Spring Boot
REST API
JSON
JWT Authentication

Purpose
This project is created for learning, practice, and backend API testing demonstration purposes.
